Home Automation Embedded Firmware

InTouch Home Automation

TODO INSERT INTERMATIC LOGO HERE

Project Description

InTouch was a home automation product that used Z-Wave, a low power wireless mesh network chip and protocol that is similar to Zigbee. InTouch products can be used to connect lights, outlets, dimmers, and thermostats to timers, handheld remotes, and motion sensors.

InTouch devices were the first Z-Wave devices that could be upgraded over the air using the Z-Wave network.

Technology Summary

Outlets and switches ran firmware directly on the 8051-based Z-Wave ASIC. More complex devices used the ASIC as a modem and used an Atmel AVR microcontroller as the main controller.

Several development and software testing tools were created using C# .NET, including a Z-Wave protocol analyzer that reverse engineered portions of the proprietary Z-Wave protocol.

  • Embedded development using C (Keil uVision and gcc).
  • Utility applications created using C#.NET.
